# Dr. Prakash Sheelvanthmath Portfolio

This is a premium, high-performance portfolio website built with React, Tailwind CSS, and Vite.

## Deployment to GitHub Pages

1. Push this code to a GitHub repository.
2. The included GitHub Action (`.github/workflows/deploy.yml`) will automatically build and deploy the site.
3. Ensure your GitHub repository settings under **Settings > Pages** is set to "GitHub Actions" as the source.

## Technical Stack
- React 19
- Tailwind CSS (via CDN)
- Vite 6
- TypeScript
